# Resume Generator

## Overview

Resume Generator is a Python based application that allows users to create, manage, search, update, and generate professional resumes in Word (.docx) format through a menu driven interface.
The project is designed to simplify resume creation by collecting user information, validating inputs, and automatically generating a well structured resume document.
---
## Features

- Create Resume
- View All Resumes
- Search Resume by Name
- Update Resume Details
- Add Multiple Education Qualifications
- Generate Resume in Word (.docx) Format
- Input Validation for User Details
- Store and Manage Multiple Resumes
- Automatically Generate Professional Personal Profile Summaries
  
---

## Technologies Used

- Python
- Object Oriented Programming (OOP)
- python docx
- Lists
- Dictionaries
- Loops and Conditional Statements
  
---

## How It Works

1. Create a resume by entering personal and educational details.
2. Store multiple resumes during program execution.
3. Search resumes by name.
4. Update existing resume details.
5. Generate a professional Word (.docx) resume automatically.
6. View all created resumes through the menu system.
